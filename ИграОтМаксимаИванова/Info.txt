Инструкция по запуску и игре:
Здравствуйте, я Иванов Максим Ивнаович и я сделал эту игру, для начала игры нужно запустить
локальный сервер т. к. код взаимодействует с .json файлами, для этого есть Host.py который можно
просто запустить и он запустит игру, в начале есть выбор для игры нужно выбрать Барона Мюнхгаузена, 
но если выбрать Государственный Эрмитаж или Академия дополнительного профессионального образования
то мы перейдём на соотвествующий сайт, а если на Иванова Максима, то мы перейдём на мою страницу
во "ВКонтакте". Далее в игре Барон предлагает нам отгадывать правду ли он говорит, для этого есть
2 кнопки в низу.

Про файлы:

Папки:
Img - с фотографиями
Html - со страницей (game.html - страница основной игры)

Файлы:
JavaScript: 
Game.js - скрипты для игры
Script.js - скрипты для меню

CSS:
Style.css - стили для меню
Style_Game.css - стили для игры

HTML:
Index.html - страница с доступными контактами

Json:
Riddle.json - файл со историями

Python:
Host.py - локальный сервер
